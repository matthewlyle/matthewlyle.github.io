Mad Dog Cafe is a delicious place to hang out and enjoy some coffee and treats in Riverdale, Toronto.

They needed a website as simple and minimal as their menu that made it easy for both their employees to update and their prospective customers to use for information.

Gallery

Fully responsive

People can be coming to this site from anywhere using any device so it was important to keep all resolutions in mind when developing. The entire site is fluid, adapting to the width of your browser so that the user can always see what's important.

Built with WordPress

The site was built using WordPress because employees had experience with it in the past and it was easy to customize to their needs exactly.

A website isn't useful if it's out of date, so we used Advanced Custom Fields to set up sections of their website just for them. This included a special menu area that made it easy for any employee to update.
